# Currency-Converter

🌐💱 Excited to share my latest project : "Currency Converter" built using HTML, CSS, Bootstrap , and JavaScript, powered by APIs!

This project showcases how I utilized an API to retrieve real-time exchange rates, enabling users to convert currencies seamlessly. The clean design and responsive layout ensure a great user experience!


🔍 What is an API?
An API (Application Programming Interface) is a set of protocols that allows different software applications to communicate. In this project, I used an API to fetch live data, making it possible to display up-to-date currency conversions without manual updates.
